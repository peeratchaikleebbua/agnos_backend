## Agnos Backend

This is project for Agnos Backend

Frontend link => https://github.com/peeratchaikleebbua/agnos.git

## Installation

```bash
# clone
$ git clone https://github.com/peeratchaikleebbua/agnos_backend.git
# install dependency
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
